Properties:
Name Type Description
id string id will be assigned to checkbox[id] & label[for].
markRequired string will add *.
fieldLayout string can be 'row' or 'column'.
styles object styles will be applied to button tag.
fieldGroup Array.<object> pass all checkboxes details as each item.
groupLayout string layout of items provided in fieldGroup can be 'row' or 'column'.
name string will be common to all checkboxes , no need to provide individually , but you can override.
validations object Does apply validations to component only 'isRequired' present.
fieldLabel string label to component field.
labelClass string | Array.<string> will be applied to TFCheckboxField label wrapper.
compClass string | Array.<string> will be applied to TFCheckboxField outermost div.
controlClass string | Array.<string> will be applied to div wraper of all checkboxes.
render function this function will run when the component is generated but not yet returned.
listeners object is an object where all listener handlers can be written as key value pair.

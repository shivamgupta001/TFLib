TFContainer

new TFContainer()

Properties:
Name Type Description
id string id will be assigned to container.
layout string can be 'row' or 'column'.
styles object styles will be applied to button tag.
tagName string can pass tag name of which container can be formed.
flex number will define what %age of width to take in comparison to others.
compClass string | Array.<string> will be applied to outermost div of component.
innerHTML string can pass html directly.
innerText string can pass text.
render function this function will run when the coponent is generated but not yet appended.
listeners object is an object where all listener handlers can be written as key value pair.

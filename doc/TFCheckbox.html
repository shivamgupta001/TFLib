Properties:
Name Type Description
id string id will be assigned to checkbox[id] & label[for].
fieldLayout string can be 'row' or 'column'.
styles object styles will be applied to button tag.
attributes object attributes will be applied to button tag.
labelClass string | Array.<string> will be applied to TFCheckbox label.
compClass string | Array.<string> will be applied to TFCheckbox.
fieldLabel string field label value.
render function this function will run when the coponent is generated but not yet appended.
listeners object is an object where all listener handlers can be written as key value pair.

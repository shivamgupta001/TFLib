Class: TFTextAreaField

TFTextAreaField

new TFTextAreaField()

Properties:
Name Type Description
id string id will be assigned to outer component.
innerId string will be assigned to input[id] & label[for].
fieldLayout string can be 'row' or 'column'.
styles object styles will be applied to outer div of component.
displayLabel string 'none' will hide display part of component.
attributes object attributes will be applied to textarea tag.
validations object Does apply validations to component only 'isRequired' present.
fieldLabel string label to component field.
labelClass string | Array.<string> will be applied to label wrapper.
compClass string | Array.<string> will be applied to outermost div.
controlClass string | Array.<string> will be applied to div wraper of all teaxtarea wrapper.
render function this function will run when the component is generated but not yet returned.
listeners object is an object where all listener handlers can be written as key value pair.
